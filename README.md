# Well Well (Todo App)

Mobile application for task management with support for recurring tasks and smart scheduling.

Built with React Native and Expo, the app focuses on improving productivity by automating repetitive tasks and handling overdue items.

## 🚧 Status
In active development

## 🛠 Tech Stack
- React Native
- Expo
- TypeScript

## ✨ Features

- Create, edit, and delete tasks    
- Support for recurring tasks (e.g. every 7 days)  
- Automatic rescheduling after completion  
- Handling overdue and upcoming tasks 
- Task prioritization using weight based on complexity  
- Local data persistence using AsyncStorage  

## 💡 Key Idea

The core feature of the app is smart recurring tasks:

When a task is completed, the next execution date is automatically calculated based on the recurrence rule (e.g. +7 days), allowing users to manage repeating activities without manual rescheduling.
